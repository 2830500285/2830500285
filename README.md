## Profile

I build AI-agent systems, evaluation harnesses, and applied research tools. My public work centers on one clear direction: making AI workflows more executable, measurable, and useful in real projects.

我关注的主线是 Agent runtime、模型评测、数学建模、数据实验和工程化落地。相比展示概念，我更倾向于把项目做成可以运行、可以验证、可以复现的系统。

## Current Focus

| Direction | What it means in my repos |
| --- | --- |
| Agent runtime | `omni-agent` explores verification-native local coding agents with eval gates, memory, subagents, and model profiles. |
| Agent evaluation | `swarm-eval-lab`, `agent-eval-learning`, and `Harness-Learning` organize practical eval methods, benchmarks, and handbook-style knowledge. |
| Applied AI research | Traffic-flow forecasting, digital comparative-advantage trade research, and mathematical-modeling projects turn models into reproducible experiments. |
| Product engineering | TypeScript, Vue, and Python projects cover coding agents, campus navigation, personal tools, and GitHub Pages work. |
